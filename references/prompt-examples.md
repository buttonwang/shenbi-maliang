## 七、组件使用

### 7.1 调用按钮组件

```
在页面中使用以下按钮样式：
- 主要操作使用 btn-primary
- 次要操作使用 btn-secondary
- 危险操作使用 btn-danger
- 取消操作使用 btn-ghost
- 小尺寸使用 btn-sm
- 表格操作列使用 btn-icon-only
```

### 7.2 使用统计卡片

```
使用 stat-card 组件展示统计数据：
- 顶部：图标和背景色
- 中部：大字号数值
- 底部：趋势指示（上升绿色箭头/下降红色箭头）
```

### 7.3 使用状态徽章

```
使用 badge 组件展示状态：
- badge-pending（黄色，待处理）
- badge-following（蓝色，进行中）
- badge-dealed（绿色，已完成）
- badge-abandoned（灰色，已取消）
```

## 八、技巧

### 8.1 指定数据存储

```
使用 LocalStorage 持久化数据：
- customers: 客户数据
- orders: 订单数据
- settings: 用户设置
- 命名格式：{模块}_{版本号}
```

### 8.2 指定图表类型

```
使用 Chart.js 时指定图表类型：
- type: 'bar' 柱状图
- type: 'line' 折线图
- type: 'pie' 饼图
- type: 'doughnut' 环形图
- type: 'radar' 雷达图
```

### 8.3 指定暗色配色

```
暗色模式使用 slate 色系：
- 背景：slate-900 (#0f172a)
- 卡片：slate-800 (#1e293b)
- 边框：slate-700 (#334155)
- 文字主色：slate-100 (#f1f5f9)
- 文字次色：slate-400 (#94a3b8)
- 强调色：blue-400 (#60a5fa)
```

## 十、业务管理系统参考案例

### 10.1 接待业务管理系统

**参考文件**：`examples/接待业务管理系统-演示.html`

**功能结构**：
```
接待业务管理系统
├── 仪表盘
│   ├── 统计卡片（今日接待、本月统计、待接待、总人次）
│   ├── 30天趋势柱状图
│   ├── 状态分布环形图
│   └── 近期接待列表
│
├── 接待记录
│   ├── 搜索栏 + 状态筛选
│   ├── 记录表格（访客姓名、单位、人数、日期、状态）
│   ├── 添加/编辑弹窗表单
│   └── 删除确认弹窗
│
├── 接待方案
│   ├── 方案类型卡片（政府来访/企业商务/媒体采访/其他）
│   ├── 生成的方案展示
│   ├── 编辑方案对话框
│   └── 方案模板列表（查看/编辑按钮）
│
├── 数据看板
│   ├── 月度趋势折线图
│   ├── 接待方式分布饼图
│   ├── 状态分布柱状图
│   └── 数据汇总卡片
│
└── 系统设置
    ├── 深色模式切换
    ├── 数据导出（JSON）
    ├── 数据导入
    └── 清空数据
```

**数据模型**：
```javascript
{
  id: string,
  visitorName: string,      // 访客姓名
  visitorCompany: string,    // 来访单位
  visitorCount: number,     // 来访人数
  visitDate: string,        // 来访日期 (YYYY-MM-DD)
  visitTime: string,        // 来访时间 (HH:mm)
  visitPurpose: string,     // 来访目的
  receptionMethod: string,   // 接待方式 (meeting-room/reception-room/tea-break/banquet)
  hostPerson: string,       // 接待人
  contactPhone: string,     // 联系电话
  status: string,           // 状态 (pending/completed/cancelled)
  remark: string,           // 备注
  createdAt: string         // 创建时间
}
```

**参考此案例生成**：
```
参考接待业务管理系统的风格，生成一个会议预约管理系统：
- 仪表盘：今日预约、会议室使用率
- 预约管理：会议室、预约人、时间段、会议主题
- 会议室管理：会议室列表、设备配置
- 数据统计：使用率图表
- 深色模式 + 移动端适配
```
