# 神笔马良 - 业务助手技能

> 说出你的需求，神笔为你实现

## 功能简介

神笔马良是一个帮助非技术背景的企业员工将业务需求转化为可用的网页应用的智能助手。只需描述需求，即可生成完整的产品文档和可运行的企业级后台管理系统。

### 核心能力

- **需求理解**：快速理解业务需求，识别核心问题和目标
- **文档生成**：基于标准 PRD 模板生成结构化的产品需求文档
- **应用开发**：生成标准的企业后台管理布局（侧边栏 + 主内容区）
- **持续迭代**：根据反馈持续优化产品

### 支持的业务场景

- 客户管理 / 订单管理 / 数据统计
- 预约系统 / 排班管理 / 工单管理
- 任何需要后台管理页面的业务场景

## 安装方法

### 方法一：解压到 skills 目录（推荐）

1. 解压下载的 ZIP 文件
2. 将解压后的 `神笔马良` 文件夹移动到以下目录：

   **Windows:**
   ```
   C:\Users\你的用户名\.mavis\skills\
   ```

   **macOS / Linux:**
   ```
   ~/.mavis/skills/
   ```

3. 重启 MiniMax Code 或开启新对话，输入 `/神笔马良` 即可使用

### 方法二：使用命令行安装

如果你使用 Git 管理配置，可以将文件夹克隆到 skills 目录：

```bash
# Windows PowerShell
git clone <仓库地址> "$env:USERPROFILE\.mavis\skills\神笔马良"

# macOS / Linux
git clone <仓库地址> ~/.mavis/skills/神笔马良
```

---

## 使用方法

1. 在 MiniMax Code 中输入 `/神笔马良` 或直接描述你的需求
2. 例如：
   - "我需要一个客户管理工具"
   - "帮我做个订单统计页面"
   - "我想做一个报名表单"
3. 神笔马良会自动生成 PRD 文档和可运行的 HTML 应用

## 注意事项

- 生成的 HTML 应用使用 LocalStorage 存储数据，适合单机使用
- 如需多设备同步或后端支持，需要额外开发
- 应用基于 Tailwind CSS CDN，无需安装任何依赖
